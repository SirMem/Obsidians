# System.out.println
System.out.println("Hello from Java!");
#### 解释

- **`System.out`**：这是Java标准库中的一个静态成员，代表标准输出流，通常是控制台。
- **`println`**：这是`PrintStream`类中的一个方法，用于向标准输出流打印一行文本，并在行末自动添加一个换行符。
