# 定义
哈希表是一个通过建立键 key 与值 value 之间的映射，实现高效的元素查询的数据结构
![[BAvMXOSKbrXnYZww-766382f6-35f5-486c-0948-15fb345814d3.png|688]]

# 桶
哈希表是基于数组实现的，哈希表中每一个元素称为桶，桶内存有键和值。
